Description

This is a file which includes instances / individuals that resulted from an exemplary mapping of tensile test data to the tensile test ontology (TTO) module of the PMD Core Ontology (PMDco). It was created in the frame of the joint research project Plattform MaterialDigital (PMD).
The file is supposed to provide a best practice example of how to use the TTO and connect data thereto (data mapping). The file may directly be stored in a triple story to be queried using SPARQL.
